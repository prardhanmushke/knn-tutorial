# k-Nearest Neighbours (k-NN) Classification Tutorial

## Overview
Complete tutorial on k-NN algorithm implementation using the Iris dataset. Includes theory, code, visualization, and analysis.

2. Run all cells (Ctrl+F9)

**OR run locally:**
```bash
